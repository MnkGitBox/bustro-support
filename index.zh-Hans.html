常见问题

什么是实时巴士跟踪，如何使用？

实时巴士跟踪自动监控您选定的巴士到站时间并实时更新，即使您最小化应用程序也能正常工作。

如何启用：

  1. 前往 设置 > 偏好设置 > 自动跟踪巴士到站（开启）
  2. 当您查看任何巴士站详情时，跟踪自动开始
  3. 您也可以手动为特定站点开启跟踪

优势：持续监控，电池优化更新，和智能后台刷新。

如何自定义附近站点的搜索半径？

自定义您的搜索区域：

  1. 从主菜单打开设置
  2. 前往偏好设置
  3. 调整"搜索半径"从100米到1000米
  4. 更改立即应用到您的附近站点

提示：城市区域使用较小半径（100-300米），郊区使用较大半径（500-1000米）。

为什么我看不到附近的巴士站？

解决方案：前往设置 > 隐私与安全 > 定位服务 > Bustro并选择"使用应用期间"来启用定位权限。

其他检查：

  • 确保您的搜索半径不会太小（检查设置 > 偏好设置）
  • 尝试下拉附近站点列表来刷新您的位置
  • 检查您是否在有巴士站的区域（仅限新加坡）

注意：Bustro 仅在应用打开时使用您的位置来显示附近站点和计算步行距离。

巴士时间准确吗？

答案：是的！我们的数据直接来自新加坡陆路交通管理局（LTA）DataMall API v3 的实时信息。时间可能因交通状况略有变化，但我们显示的是最新可用信息。

数据包括：

  • 车辆类型指示（单层、双层、铰接巴士）
  • 无障碍通道状态
  • 乘客负载（有座位、有站位、站位有限）
  • 精确的到站预测（"即将到站"、"2分钟"等）

刷新频率：您可以在设置中自定义刷新间隔，从15秒到2分钟。

如何将巴士站添加到收藏夹？

步骤：

  1. 点击任何巴士站打开详情
  2. 点击心形图标（♡）添加到收藏夹
  3. 用个人名称、图标和颜色进行自定义
  4. 从"收藏夹"标签快速访问

高级功能：

  • iCloud 同步：收藏夹自动在所有设备间同步
  • 视觉组织：选择自定义颜色和图标便于识别
  • 重新排序：拖放来组织您的收藏夹列表
  • 地图集成：收藏夹在地图上以自定义样式显示

天气小工具有什么作用？

功能：天气小工具使用 Apple WeatherKit 显示当前天气状况，帮助您规划出行。了解是否下雨或晴朗有助于决定何时出发和带什么物品。

显示信息：

  • 当前温度和天气状况
  • 每小时降水概率
  • 基于位置的天气（您当前位置或新加坡默认）
  • 自动15分钟更新

提示：点击天气小工具打开完整的天气应用以获取详细预报和雷达。

如何访问和使用地铁地图？

访问：从主界面点击地铁按钮以查看新加坡完整的地铁网络地图。

功能：

  • 互动查看：平移和缩放来探索整个网络
  • 高分辨率：清晰的地图，包含所有站点和线路
  • 图例：所有地铁线路和站点的综合图例
  • 集成：寻找地铁站附近的巴士站以实现无缝换乘

出行规划：完美适用于规划结合巴士和地铁的多模式新加坡出行。

iCloud 同步如何与我的收藏夹一起工作？

自动同步：启用后，您的收藏夹、自定义名称、颜色和图标会自动在您所有的 Apple 设备间同步。

设置：

  1. 确保您在所有设备上都登录了 iCloud
  2. 收藏夹在后台自动同步
  3. 更改会在几分钟内出现在其他设备上

同步问题故障排除：

  • 在 iOS 设置 > [您的姓名] > iCloud 中检查 iCloud 设置
  • 确保您有互联网连接
  • 尝试开关飞行模式来刷新同步

故障排除

实时巴士跟踪不工作或耗电？

常见解决方案：

  1. 检查设置：确保在设置 > 偏好设置中启用"自动跟踪巴士到站"
  2. 手动切换：尝试手动为特定巴士站启用跟踪
  3. 后台刷新：在 iOS 设置中为 Bustro 启用后台应用刷新
  4. 互联网连接：跟踪需要有效的互联网连接

电池优化：

  • 跟踪设计为电池高效，采用智能刷新间隔
  • 如果电池使用量是问题，考虑在设置中降低刷新频率
  • 不需要时跟踪会自动暂停

应用崩溃或运行缓慢

快速修复：

  1. 强制关闭应用并重新打开
  2. 重启您的iPhone
  3. 确保您有最新的应用版本
  4. 检查可用存储空间（至少需要1GB空闲空间）

巴士时间不更新

解决方案：

  • 检查您的互联网连接（WiFi 或蜂窝数据）
  • 尝试在 WiFi 和蜂窝数据之间切换
  • LTA 偶尔会维护 - 时间会自动恢复

地图无法正常加载

故障排除：

  • 确保已启用定位服务
  • 检查互联网连接
  • 尝试缩小再放大
  • 如果地图显示空白，请重启应用
